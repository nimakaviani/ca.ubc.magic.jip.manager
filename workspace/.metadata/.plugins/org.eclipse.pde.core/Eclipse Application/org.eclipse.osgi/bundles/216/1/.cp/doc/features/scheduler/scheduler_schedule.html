Schedule Wizard

A task can be scheduled to start:

  • at a particular time on a particular date,
  • at Start Up when Application Developer/Eclipse is first started,
  • at Shut Down when Application Developer/Eclipse is exited,
  • after any other arbitrary task,
  • before a selected class is launched,
  • only when triggered manually

A task can be scheduled to repeat after a specified number of days, hours, and/or minutes have elapsed since the last time that the task was started.  The minimum number of minutes between repetitions is 5 minutes. Optionally, the scheduler can prompt the user before a task is run.  The prompter can be configured to only appear for a specified number of seconds and if no response, then either automatically run the task or not.

Click the "Next >" button to give the task a name and a description.
