Export Set - EAR File

The "Add Export Operation" wizard pages for the "EAR file" operation are identical to those one would find in the standard export wizard if one were starting a EAR file export manually. The only difference is that the "Finish" button is used to save the export operation rather than execute it immediately.

An codepro.exportEARToEarFile Ant task is used to record the options chosen and perform the operation as part of an Export Set.
