Java Applet Wizard

The Java Applet Wizard allows you to create new Java applets subclassed from either java.applet.Applet or javax.swing.JApplet. New Applets can be created via the standard New wizard by selecting the "Java" category, or via the popup "New" menu or the icon on the main toolbar. Applets can also be created by selecting the   Applet pattern in the Java Pattern Wizard.
