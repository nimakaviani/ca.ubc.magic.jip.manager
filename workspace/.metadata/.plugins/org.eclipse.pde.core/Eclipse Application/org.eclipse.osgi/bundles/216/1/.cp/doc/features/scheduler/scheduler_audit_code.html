Schedule Audit Code

To schedule a code audit report to be run at a specified time, open the scheduler view, right click to access the popup menu, select the "New" submenu, and then select "Audit Code". The default audit rule set may be selected as well as the collection of resources or working set that should be audited. Once the audit results have been collected, the results may either be displayed in the audit view or written out to a report in a variety of formats including text, tab and comma delimited, HTML and XML. The results may also be grouped by audit rule or resource.

Click the "Next >" button to schedule when the exit is to occur.
