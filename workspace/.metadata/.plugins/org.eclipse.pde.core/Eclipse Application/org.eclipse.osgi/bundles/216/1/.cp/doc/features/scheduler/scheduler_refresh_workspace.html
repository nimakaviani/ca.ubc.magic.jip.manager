Schedule Refresh Workspace

To schedule a refresh of the workspace (similar to what the -refresh command line option does) to be run at a specified time, open the scheduler view, right click to access the popup menu, select the "New" submenu, and then select "Refresh Workspace". In addition to refreshing the workspace, it will also scan for and recognize and new Eclipse projects that were created externally to Eclipse (either by hand or via VA Assist when Eclipse was not running).

Click the "Next >" button to schedule when the refresh is to occur.
