Modified Types

The CodePro Modified Type view tracks all modifications or additions made to various types within the workspace. The types may be deleted or sorted by name, package, time of modification or type of modification (e.g., added or modified). Selecting a type in the list will open an editor and focus it on the code for the type.

Options are provided via a preference page to show only the most recent n number of types and/or only those types modified in the specified time period (e.g., the last week, the last n days, etc.).

Early Startup

In order that CodePro track modified types, this plugin must be loaded and executed when Eclipse is launched.
