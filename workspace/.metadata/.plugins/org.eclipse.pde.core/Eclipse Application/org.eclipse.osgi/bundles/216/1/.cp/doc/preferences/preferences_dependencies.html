Preferences - Dependencies

The View Options section is used to control the CodePro Dependency Analysis view options.

External color

This options determines the color of the external element nodes.

Internal color

This options determines the color of the internal element nodes.

Strongly Connected color

This options determines the color of the strongly connected element nodes.

Background color

This options determines the background color of the analysis.

Dependency line style

This options determines the type of connection lines used between nodes in the graph. Straight lines connect nodes by the shortest possible path. Manahattan lines connect nodes using orthogonal line segments running either horizontally or vertically (with one or more 90 degree angles between segments).

The HTML Report Options section is used to control the CodePro Dependency Analysis HTML report options.

Include project section (when available)

This options determines whether project-level analysis is included.

Include package section

This options determines whether package-level analysis is included.

Include type section

This options determines whether type-level analysis is included.

Include explanatory section

This options determines whether an explanatory section is included.

The HTML Report Graph Options section is used to control the report graphing options.

Include dependency graphs

This options determines whether any graphs are included in the report. Each graph is stored as a separate GIF file.

Include external nodes in graphs

This options determines whether external nodes are shown in the graphs.
