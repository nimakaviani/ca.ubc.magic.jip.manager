Preferences - Startup

This page is used to control various startup options.

Refresh workspace and scan for new projects

This option schedules a special system task to run at startup that will refresh the workspace (similar to what the -refresh command line option does). In addition to refreshing the workspace, it will also scan for and recognize new Eclipse projects that were created externally to Eclipse.

Show CodePro menu

This option determines whether the CodePro menu is added to the main menu bar. The CodePro menu provides convenient access to many CodePro functions (i.e., perspectives and views), help and on-line web and e-mail links.
