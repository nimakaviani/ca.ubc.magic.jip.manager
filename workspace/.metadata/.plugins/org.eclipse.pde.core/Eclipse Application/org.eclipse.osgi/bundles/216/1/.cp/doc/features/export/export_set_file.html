Export Set - File System

The "Add Export Operation" wizard pages for the "File system" operation should be identical to those one would find in the standard export wizard if one were starting a file system export manually. The primary difference is that the "Finish" button is used to save the export operation rather than execute it immediately.

Since export sets are saved as Ant scipts, an "Insert Variable" button has been added that allows the user to insert a variable into the export destination directory field.

An codepro.exportToFileSystem Ant task is used to record the options chosen and perform the operation as part of an Export Set.
