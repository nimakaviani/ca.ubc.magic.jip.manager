Export Set - EJB File

The "Add Export Operation" wizard pages for the "EJB file" operation are identical to those one would find in the standard export wizard if one were starting a EJB file export manually. The only difference is that the "Finish" button is used to save the export operation rather than execute it immediately.

An codepro.exportEJBToJarFile Ant task is used to record the options chosen and perform the operation as part of an Export Set.
